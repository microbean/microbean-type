microBean™ Type

The microBean™ Type project provides classes and interfaces assisting with Java Type instances.

Requirements

microBean™ Type requires a Java runtime of version 11 or higher.

Installation

microBean™ Type is available on Maven Central. Simply include microBean™ Type as a Maven dependency:

<dependency>
  <groupId>org.microbean</groupId>
  <artifactId>microbean-type</artifactId>
  <!-- Always check https://search.maven.org/artifact/org.microbean/microbean-type for up-to-date available versions. -->
  <version>0.0.9</version>
</dependency>

Documentation

Full documentation is available at microbean.github.io/microbean-type.
