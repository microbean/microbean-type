microBean™ Type

The microBean™ Type project provides classes and interfaces assisting with Java Type instances, as well as other frameworks’ representations of such instances.

Status

This project is currently experimental, in a pre-alpha state, and unsuitable for production use.

Compatibility

Until further notice, this project’s APIs are subject to frequent backwards-incompatible signature and behavior changes, regardless of project version and without notice.

Requirements

microBean™ Type requires a Java runtime of version 17 or higher.

Installation

microBean™ Type is available on Maven Central. Include microBean™ Type as a Maven dependency:

<dependency>
  <groupId>org.microbean</groupId>
  <artifactId>microbean-type</artifactId>
  <!-- Always check https://search.maven.org/artifact/org.microbean/microbean-type for up-to-date available versions. -->
  <version>0.3.2</version>
</dependency>

Documentation

Full documentation is available at microbean.github.io/microbean-type.
